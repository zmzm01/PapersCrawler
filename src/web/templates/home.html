PapersCrawler automatically tracks 21 journals from 7 publishers through an 8-phase pipeline — discovery, metadata enrichment, page scraping, relevance filtering, PDF parsing, structured summarization, report generation, and email delivery.

Pipeline Phases

Quick Guide

Page What you can do Notes
Pipeline Run individual phases, watch live logs with level filter, reset phase states Config SKIP toggles disable Pipeline buttons. Reset shows impact details.
Papers Browse papers, sort by created or published date. Semantic scores shown if Phase D ran; LLM relevance status shown for all.
Report Select papers with LLM summaries, generate Markdown reports, preview and download Only papers with llm_summary_status = success appear. Use checkboxes to select.
Data Sources Enable or disable journals and their RSS / CrossRef data sources per-journal Saved to separate override file, does not modify publishers.yaml.
Subscriptions Manage email subscribers, add/remove/test, import from .env Active subscribers receive the daily report when Phase H runs.
Logs View pipeline log file, filter by severity level Shows most recent ~200 KB. New log lines via SSE.
Config Toggle SKIP switches, edit domain description, test connectivity, edit YAML configs SKIP toggles affect Pipeline buttons; CLI unaffected.
YAML editors validate syntax before saving.
