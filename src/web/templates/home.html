Quick Guide

Page What you can do Notes
Pipeline Run individual phases, watch live logs with level filter, reset phase states, view progress charts Buttons always run phases regardless of SKIP_PHASE config. Reset shows confirmation with impact details.
Papers Browse papers sorted by semantic similarity to your research domain Only shows papers that have been processed by Phase D (Semantic Filter).
Report Select papers with LLM summaries, generate Markdown reports, preview in-browser and download Only papers with llm_summary_status = success appear in the list. Use checkboxes to select specific papers.
Logs View the pipeline log file (data/PaperCrawler.log), filter by severity level Shows the most recent ~200 KB. New log lines arrive in real time via SSE.
Config Toggle SKIP switches (persisted to data/skip_overrides.json), edit publishers.yaml and keywords.yaml SKIP toggles affect CLI behavior; Web UI ignores them (click Run to execute).
